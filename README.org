* Arch setup (omarchy)

1. Install emacs-wayland
2. Symlink hypr

#+begin_src sh
ln /home/nick/dotfiles/.config/hypr ~/.config/
#+end_src

3. Symlink doom

#+begin_src sh
ln /home/nick/dotfiles/.config/doom ~/.doom.d
#+end_src

4. Install doom emacs
5. Add **amdgpu.dcdebugmask=0x200** to kernel params // prevents suspend from breaking
6. Reset waybar when pluggin in external monitor

#+begin_src sh
pkill waybar && hyprctl dispatch exec waybar
#+end_src

For managing power profiles

#+begin_src sh
powerprofilesctl set balanced # performance, power-saver, balanced
#+end_src
