* Arch setup (omarchy)

1. Install packages
2. Symlink hypr

#+begin_src sh
ln /home/nick/dotfiles/.config/hypr ~/.config/
#+end_src

3. Symlink doom

#+begin_src sh
ln /home/nick/dotfiles/.config/doom ~/.doom.d
#+end_src

4. Install doom emacs

** Notes

*** Reset waybar when pluggin in external monitor

#+begin_src sh
pkill waybar && hyprctl dispatch exec waybar
#+end_src

*** For managing power profiles

#+begin_src sh
powerprofilesctl set balanced # performance, power-saver, balanced
#+end_src

*** Apple superdrive, use vlc to eject

#+begin_src sh
yay -Sy sg3_utils
sg_raw --cmdset=1 /dev/sr0 EA 00 00 00 00 00 01
#+end_src

*** Keyboard

#+begin_src sh
asusctl aura static -c ffffff # set keybaord color to white
#+end_src

*** Keyboard brightness

#+begin_src sh
# install xorg-xev
pacman -S xorg-xev
xev | grep keysym
#+end_src
