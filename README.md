# Setup 

1. link .config from home dir `ln -s dotfiles/.config .config`

2. link tmux config from home dir `ln -s dotfiles/.tmux.conf .tmux.conf`

3. set cambell theme in wsl, follow doom emacs install for wsl then run: 

```
~/.emacs.d/bin/doom install
```

NOTE: if using repos under windows directory, use git.exe installed in windows and enable in zshrc
